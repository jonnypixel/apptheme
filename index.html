Frequently Asked Questions (FAQ)

Which versions of Joomla are supported?

Currently AppTheme only supports Joomla 3.0+ since the components use Bootstrap. We’re considering a Joomla 2.5 version if we can find a way to deliver a good user experience.

What if I want to keep my current template & use only the tablet or phone version?

Sweet! We thought of that. You can use the free plugin to load AppTheme for specific devices and browsers.

Are there any domain restrictions?

Nope. You can use this template on all your sites. The only restriction is that you can’t directly distribute or sell the templates. You can sell your services in installing them, etc.

Do I need to purchase a subscription?

No way! This is a one-time purchase that grants you access to all future versions of AppTheme.

How do I download the latest version?

Simply visit our purchase page, login, and download the file again.

Who's Behind This Craziness?

AppTheme is the love child of AdminPraise, TapTheme & Projectfork Themes, all of which are long-standing projects of Kyle Ledbetter.
